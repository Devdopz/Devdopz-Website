# Support

If you need help with Devdopz, use the right channel so the project stays easy
to maintain.

## Where to Ask for Help

### Bugs

Open a GitHub issue if something is broken and you can describe:

- what happened
- how to reproduce it
- what you expected

### Feature Requests

Open a GitHub issue with a clear proposal, use case, and expected outcome.

### General Questions

You can reach out through:

- Telegram: `https://t.me/devdopzlearningclub`
- Email: `teamdevdopz@gmail.com`

## Security Issues

For vulnerabilities or sensitive security concerns, do not use public issues.

Please follow [SECURITY.md](./SECURITY.md).
